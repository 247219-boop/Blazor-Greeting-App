# Blazor Greeting App

## 📌 Project Description
This is a Blazor Web App that demonstrates:
- Two-way data binding using @bind
- One-way data binding using computed properties
- Component-based architecture
- Navigation menu system

---

## 📌 Pages

### 1. Greeting Page
- Takes user input (name)
- Displays dynamic greeting message

### 2. Home Page
- Default landing page

### 3. Counter Page
- Simple increment counter

### 4. Weather Page
- Sample data page
